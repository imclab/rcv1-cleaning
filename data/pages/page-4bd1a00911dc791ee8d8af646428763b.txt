(function($){ 
     $.fn.extend({  
         tabify: function( callback ) {
         	
			function getHref(el){
				hash = $(el).find('a').attr('href');
				hash = hash.substring(0,hash.length-4);
				return hash;
			}
			
		 	function setActive(el){
		 		
				$(el).addClass('active');
				$(getHref(el)).show();
				$(el).siblings('li').each(function(){
					$(this).removeClass('active');
					$(getHref(this)).hide();
				});
			}
			
			return this.each(function() {
			
				var self = this;
				var	callbackArguments 	=	{'ul':$(self)};
					
				$(this).find('li a').each(function(){
					$(this).attr('href',$(this).attr('href') + '-tab');
				});
				
				function handleHash(){
					
					if(location.hash && $(self).find('a[href=' + location.hash + ']').length > 0){				
						setActive($(self).find('a[href=' + location.hash + ']').parent());
					}
				}
				
				if(location.hash){
					handleHash();
				}
					
				setInterval(handleHash,100);
				
				$(this).find('li').each(function(){
					if($(this).hasClass('active')){
						$(getHref(this)).show();
					} else {
						$(getHref(this)).hide();
					}
				});
				
				if(callback){
					callback(callbackArguments);
				}	
				
            }); 
        } 
    }); 
})(jQuery);
	
</script>

<h3>WELCOME TO JAE</h3>
<p>Welcome to JAE, the Japanese Auto Extravaganza &#8211; the UK&#8217;s largest gathering of Japanese cars, clubs and scene.</p>
<p>Held annually at the fantastic <strong>Wicksteed Park, </strong><strong>Northamptonshire</strong>. For more information on Wicksteed Park or to find how to get there go to their website: <strong><span style="text-decoration: underline;"><span style="color: #000080; text-decoration: underline;"><a title="Wicksteed Park" href="http://www.wicksteedpark.co.uk/" target="_blank"><span style="color: #000080; text-decoration: underline;">http://www.wicksteedpark.co.uk/ </span></a></span></span></strong></p>
<p>JAE is unlike any other Japanese car show you have been to.  Described by <em>Japanese Performance</em> as<span style="color: #ff0000;"><strong> &#8220;The biggest and best club event of the year &#8230;.. there&#8217;s just something special about JAE&#8221;</strong></span></p>
<p>JAE is a full long weekend extravaganza of camping, entertainment and activities which embraces all Japanese cars from the UK and Europe. Starting on Thursday evening and rolling all the way through Sunday JAE is a weekend of like-minded family fun.</p>
<p>Now in its 18th year, JAE continues to be the favourite event for the Japanese car scene. <em>Banzai</em> have said <span style="color: #ff0000;"><strong>&#8220;It&#8217;s the kind of weekend people hold out for, even altering holidays to attend&#8221;</strong></span></p>
<p>The JAE website is packed with all the information you need. Whether you&#8217;re a club organiser looking for the forms, help and advice you need it&#8217;s all here. If you&#8217;re a club member and want to know more, then the website will tell you everything you need to know. or if you just want to come along and see what all the fuss is about then the JAE website is the best place to start</p>
<p>We hope to see you in 2013, For 2012 ticket sales were up and car clubs were up. For 2013 we have even more space and more activities, contact us to get involved!!</p>
<p>John, Glen, Nigel and all at JAE TEAM</p>
