  function v3_setwidth_slideshow (img, s) {
    var w = $(img).width();
    $(".v3_slideshow .slide" + s + " > span").css('width', w + 8);
    $(".v3_slideshow .slide" + s + " > span .overlay").css('width', w);
    $(".v3_slideshow .slide" + s + " > .big_caption").css('width', w);

  function v3_show_slide (s) {
    $(".v3_slideshow .slides_wrapper").animate({scrollLeft: s * v3_width}, 1000);
    googletag.cmd.push(googletag.pubads().refresh());
  }
  
  function v3_offarrows_slideshow () {
    $('.v3_slideshow .arrow').removeClass('off');
    var current = $(".v3_slideshow .nav_inner_wrapper").attr('scrollLeft');
    var l = (navw * navc) + current;
    
    if (current == 0) {
      $('.v3_slideshow .arrow.left').addClass('off');
    }
    
    else if (l >= max_nav_left) {
      $('.v3_slideshow .arrow.right').addClass('off');
    }
  }
  
  function v3_navleft_slideshow () {
    var current = $(".v3_slideshow .nav_inner_wrapper").attr('scrollLeft');
    var l = current - (navw * navc);
    
    if (current > 0) {
      v3_offarrows_slideshow();
      $(".v3_slideshow .nav_inner_wrapper").animate({scrollLeft: l}, 500, 'swing', v3_offarrows_slideshow);
    }
  }
  
  function v3_navright_slideshow () {
    var current = $(".v3_slideshow .nav_inner_wrapper").attr('scrollLeft');
    var l = (navw * navc) + current;
    
    if (l < max_nav_left) {
      v3_offarrows_slideshow();
      $(".v3_slideshow .nav_inner_wrapper").animate({scrollLeft: l}, 500, 'swing', v3_offarrows_slideshow);
    }
  }
  
  function v3_next_arrows () {
    var n = v3_current + 1;
    $(".slide_nav_top .count").html('Photo ' + n + ' of ' + photo_count);
    
    if (v3_current + 1 == photo_count) {
      $('.slide_nav_top a.nav_right').addClass('end');
    }
    
    else {
      $('.slide_nav_top a.nav_right').removeClass('end');
    }
    
    if (v3_current == 0) {
      $('.slide_nav_top a.nav_left').addClass('end');
    }
    
    else {
      $('.slide_nav_top a.nav_left').removeClass('end');
    }
  }
  
  function v3_next_slide () {
    var next = v3_current + 1;
    if (next < photo_count) {
      $(".v3_slideshow .slides_wrapper").animate({scrollLeft: next * v3_width}, 1000);
      v3_current = next;
      
      v3_next_arrows();
      
      googletag.cmd.push(googletag.pubads().refresh());
    }
  }
  
  function v3_prev_slide () {
    var next = v3_current - 1;
    if (next >= 0) {
      $(".v3_slideshow .slides_wrapper").animate({scrollLeft: next * v3_width}, 1000);
      v3_current = next;
      
      v3_next_arrows();
      
      googletag.cmd.push(googletag.pubads().refresh());
    }
  }
  
  function v3_slideshow_init () {
    var hash = window.location.hash;
    if (hash) {
      v3_current = parseInt(hash.substr(-1, 1));
      $(".v3_slideshow .slides_wrapper").animate({scrollLeft: v3_current * v3_width}, 1000);
      v3_next_arrows();
    }
  }

          <p>
	With less than a week left for holiday shopping, we at CultureMap would like to offer a happy medium between street brawls in <a href="http://houston.culturemap.com/newsdetail/12-01-12-10-03-dvf-yoga-mats-lieber-compacts-popular-at-neiman-marcus-target-collection-but-not-lam-skateboard/">Target and Neiman Marcus</a> and just giving up and throwing gift cards at loved ones. How about a nice book?</p>
<p>
	The well-selected, gifted book literally and sometimes literarily tells a story to the receiver and has the ability to send many messages. I love you. I know you so well. I understand what you like. Please stop trying to force me to read <em>50 Shades of Grey</em>.</p>
<p>
	So with a little help from our favorite independent bookstores, here is the CultureMap definitive, but completely random, book gift guide.</p>
<p>
	<strong>Contemporary literary reader</strong></p>
<p>
	For the contemporary literary reader with shelves of signed copies and tales of meeting authors, pick up Michael Chabon&rsquo;s <em>Telegraph Avenue</em>. Chabon will be in Houston Jan. 28 for the first Inprint reading of the new year, so the book, a pen, and event ticket might be the perfect choice for this reader.&nbsp;</p>
<p>
	<strong>Movie lover </strong></p>
<p>
	For the movie lover who&rsquo;s just not into books, let him know Hollywood is with a gift basket filled with <em>The Life of Pi</em>, <em>Cloud Atlas</em>, <em>Les Mis&eacute;rables</em> (the book turned musical, turned movie), <em>Anna Karenina</em>, <em>Jack Reacher</em>, <em>Team of Rivals: The Political Genius of Abraham Lincoln</em>, and <em>The Hobbit</em>. Do readers&#39; imaginations project at a <a href="http://houston.culturemap.com/newsdetail/12-14-12-07-32-ithe-hobbit-an-unexpected-journeyishares-a-lot-with-ilord-of-the-ringsi-but-misses-its-magic/">48-frames-a-second</a> or 24?</p>
<p>
	<strong>Teen reader</strong></p>
<p>
	If instead you&rsquo;re shopping for a teen reader who wants to get into the next big series before the rest of the world flocks to the obligatory movie, <a href="http://www.bluewillowbookshop.com">Blue Willow Bookshop&rsquo;s</a> Valerie Koehler suggests the <em>Legend</em> series by <a href="http://marielu.org/index.html">Marie Lu</a>, who used to call Sugar Land home.</p>
<p>
	&ldquo;Post apocalyptic novels are still very hot this year. In this one, the main character is a teen who is slated to become an assassin and things turn treacherous when she is sent to kill a boy who may or may not have killed her brother. Lots of action which appeals to all teens,&rdquo; says Koehler.</p>
<p>
	<strong>Food lover</strong></p>
<p>
	Need a gift for a foodie who always buys locally? Try a cookbook by a favorite local chef. <a href="http://www.chefhugoortega.com">Hugo Ortega&rsquo;s</a> <em>Street Food of Mexico</em> is not only filled with his favorite recipes, but he includes personal stories to savor with each dish.</p>
<p>
	<strong>Fashionista</strong></p>
<p>
	If that fashionista in your life loves the behind-the-scenes drama of the industry as much as the clothes, find some sophisticated gift wrap for <em>Grace: A Memoir</em> by Grace&nbsp;Coddington, the woman who managed to steal the spotlight from Anna Wintour in not <a href="http://www.theseptemberissue.com">one</a> but <a href="http://www.hbo.com/documentaries/in-vogue-the-editors-eye/synopsis.html">two</a> documentaries about <em>Vogue</em>.</p>
<p>
	<strong>Hipster satire-lover</strong></p>
<p>
	Give the gift of historical irony to your favorite hipster satire-lover by pairing Stephen Colbert&rsquo;s latest, <em>America Again: Re-becoming the Greatness We Never Weren&#39;t</em>, with some ancient school <em>Gulliver&rsquo;s Travels</em> because if Jonathan Swift were alive today, he&rsquo;d no doubt have his own Super PAC.</p>
<p>
	<strong>Bibliophile</strong></p>
<p>
	For the bibliophile who refuses to buy an e-reader and is looking for validation, <a href="http://www.brazosbookstore.com">Brazos Bookstore</a> manager Jeremy Ellis recommends Chris Ware&rsquo;s <a href="http://www.brazosbookstore.com/book/9780375424335"><em>Building Stories</em></a>: &ldquo;An oversized box of 27 different booklets, each with a different format and point-of-view. This is a book to wade through rather than read start to finish. An eEdition would not be the same.&rdquo;</p>
<p>
	<strong>Lovable luddite</strong></p>
<p>
	And if that lovable luddite reader is raising little ones, Valerie Koehler recommends <a href="http://www.bluewillowbookshop.com/book/9780811870535"><em>In My Den</em></a> by Sara Gillingham because you can&rsquo;t download the felt finger puppet that comes with the board book and then poke it through a Kindle.</p>
<p>
	<strong>Future forecaster</strong></p>
<p>
	For the gambler, baseball stats lover, and fortune-teller in your life, let them know that the future can sometimes be forecast with <em>The Signal and the Noise: Why Most Predictions Fail but Some Don&#39;t</em>, the new book by this nation&rsquo;s recently chosen math-nerd-in-chief, <a href="http://knowyourmeme.com/memes/nate-silver-drunk-nate-silver">Nate Silver</a>.</p>
<p>
	<strong>An alternative to <em>50 Shades of Grey</em></strong></p>
<p>
	And finally for that sweet, budding-sadist friend who won&rsquo;t stop insisting you have to read <em>50 Shades of Grey</em>, Brazos Bookstore recommends <em>The Story of O</em> by Pauline Reage.</p>
<p>
	Written in 1954, &ldquo;The novel, originally published in French, is described as a story of love and submission. It won the French literary prize that year and has been in print ever since,&rdquo; says Ellis, who also suggests <em>Exit to Eden</em>, Anne Rice&rsquo;s S&amp;M romance written under her pseudonym Anne Rampling.</p>
<p>
	If those suggestions don&rsquo;t send the message, throw a hardback copy of Henry Miller&rsquo;s <em>Tropic of Cancer</em>&nbsp;your best friend&#39;s head. <a href="http://houston.culturemap.com/newsdetail/05-29-12-10-16-what-are-you-reading-carrie-fisher-loves-henry-miller-raw-language-and-hates-ereaders/">Carrie Fisher</a> would approve.</p>
