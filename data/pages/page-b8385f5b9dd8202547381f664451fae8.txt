        function AddToFavourites(pageTitle, pageUrl, callback) {
            var lnkName = typeof (pageTitle) != "undefined" ? pageTitle : jQuery("#txtlnkName").val();
            var lnkUrl = typeof (pageUrl) != "undefined" ? pageUrl : encodeURIComponent(document.URL);
            if (lnkUrl == "")
                return false;
            if (lnkName == null || lnkName.length <= 0)
                lnkName = document.getElementsByTagName("title")[0].innerHTML.replace("- Atmel Corporation", "");
            jQuery.ajax({
                Type: "GET",
                url: "https://secure.atmel.com/secure/AddFavorites.ashx",
                dataType: 'jsonp',
                data: 'lnkName=' + lnkName + '&lnkUrl=' + lnkUrl,
                timeout: 9000,
                //if response is successfull, check for authentication status
                success: function (data) {
                    var sResult = eval(data);
                    if (typeof (sResult) != "undefined") {
                        if (sResult.Status == "OK")
                            callback(true);
                        else if (sResult.Status == "ERROR")
                            callback(false);
                        else if (sResult.Status == "INVALID")
                            callback(false);
                    }
                    else callback(false);
                },
                error: function (data) {
                    callback(false);
                }
            });
        }

        /*Function to read the cookie value for given name check if cookie holds any value then return its value else return empty.*/
        function readCookie(cname) {
            if (document.cookie.length > 0) {
                cstart = document.cookie.indexOf(cname + "=");
                if (cstart != -1) {
                    cstart = cstart + cname.length + 1;
                    cend = document.cookie.indexOf("|", cstart);
                    if (cend == -1) cend = document.cookie.length;
                    return unescape(document.cookie.substring(cstart, cend));
                }
            }
            return "";
        }
        /*Function to set the cookie for given name, the cookie expires based on the value of the expiredays*/
        function setCookie(cname, value, expiredays) {
            var exdate = new Date();
            exdate.setDate(exdate.getDate() + expiredays);
            document.cookie = cname + "=" + escape(value) + ((expiredays == null) ? "" : "|expires=" + exdate.toUTCString()) + ";domain=.atmel.com;path=/";
        }

            <h2>The Atmel Automotive Commitment</h2>
            <div class="floatright">
                <div class="maintouts">
                    
                    <div class="touts">
                        
<!-- Start: Tout -->
<!--googleoff: all-->


	
	
	
	
	<a href="/products/automotive/automotive_compilation.aspx"  title="» Learn More "><img src="/Images/tout_auto_compilation.jpg"  alt="" width="190" /></a>	




    
    <p>Automotive Compilation Archive Online -- Enjoy Browsing a Broad Range of Automotive Articles</p>
    
        
        
        <a href="/products/automotive/automotive_compilation.aspx"  class="more" >» Learn More</a>        
    

<!--googleon: all-->
<!-- End: Tout -->

                            
                    </div>
                    
                </div>
            </div>
            
            
            
            <p>Automotive electronics are a demanding application, encompassing a wide range of products, from body electronics, engine components, and access systems to lighting and entertainment components. They must support small designs and offer a high level of integration, to fit the limited space available in automotive environments. To meet the diverse needs of a variety of automotive applications, developers need solutions that offer:</p><ul><li>Compliance with strict automotive qualification demands, including industry standards and automotive-qualified fabs</li><li>Flexible interface options for a variety of HMI capabilities</li><li>Standards-based connectivity for intelligent networked solutions</li><li>Protection and compatibility for high-voltage products, including robust electromagnetic compatibility (EMC); electrostatic discharge (ESD) protection; and OEM approvals</li><li>Resistance to high temperatures for trouble-free operation in automotive environments</li><li>Maximum safety for automobile access</li><li>Low power consumption for maximum efficiency</li></ul>
            
            
            
            
            <h2>A Complete Set of Solutions for Automobiles</h2>
            
            
            
            <p>Atmel® provides a broad product portfolio for automotive applications that meet the strict automotive quality demands and that make vehicles more safe, economical and convenient. Our solutions include:</p>
            
            
            <ul class="chevron">
                
                <li>
                    <a href="/applications/automotive/battery_mgmt/default.aspx" ><strong>Battery Management</strong></a>                    &mdash; 
                    Battery efficiency and lifespan are key elements for automotive battery applications. Li-ion batteries are up to 30 percent smaller and 50 percent lighter than conventional NiMH batteries, yet store considerably more energy. While Li-ion batteries offer concrete advantages in terms of size, weight, recharge speed, life span, and resistance to memory effects, they also quickly lose capacity, and are potentially unsafe if overcharged or discharged too deeply. A battery management system is required to avoid this to happen. For such systems, Atmel provides Li-ion battery management devices that offer high accuracy analog measurement functions in combination with efficient active cell balancing ensuring the optimum usage of the batteries capacity
                </li>
                
                <li>
                    <a href="/applications/automotive/car_access/default.aspx" ><strong>Car Access</strong></a>                    &mdash; 
                    The Atmel® automotive electronic efforts are not only focused on large complex electronic systems, but also on smaller independent applications. Atmel builds on decades of experience in automotive electronic designs, and offers a wide range of uniquely tailored standalone solutions.
                </li>
                
                <li>
                    <a href="/applications/automotive/car_radio/default.aspx" ><strong>Car Radio</strong></a>                    &mdash; 
                    The Atmel® automotive electronic efforts are not only focused on large complex electronic systems, but also on smaller independent applications. Atmel builds on decades of experience in automotive electronic designs, and offers a wide range of uniquely tailored standalone solutions.
                </li>
                
                <li>
                    <a href="/applications/automotive/networking/default.aspx" ><strong>Networking</strong></a>                    &mdash; 
                    The Atmel® automotive electronic efforts are not only focused on large complex electronic systems, but also on smaller independent applications. Atmel builds on decades of experience in automotive electronic designs, and offers a wide range of uniquely tailored standalone solutions.
                </li>
                
                <li>
                    <a href="/applications/automotive/microcontrollers/default.aspx" ><strong>Microcontrollers</strong></a>                    &mdash; 
                    The Atmel® automotive electronic efforts are not only focused on large complex electronic systems, but also on smaller independent applications. Atmel builds on decades of experience in automotive electronic designs, and offers a wide range of uniquely tailored standalone solutions.
                </li>
                
                <li>
                    <a href="/applications/automotive/motor_control_systems/default.aspx" ><strong>Motor Control Systems</strong></a>                    &mdash; 
                    Atmel® has more than 15 years' experience with driver ICs for DC motors. We supply products at a high-volume for a variety of common body electronic applications, including mirror control and flap control in HVACs. Although the Atmel driver ICs are equipped with a variety of types of driver stages, they all share the same protection features – short-circuit protection, temperature warning and switch off, low voltage protection, and open load detection – all of which are a must for automotive electronics.
                </li>
                
            </ul>
            
            
            
            <h2>The Atmel Advantage</h2>
            
            
            
            
            <p>Atmel has demonstrated its strong commitment to the automotive market. Our company has more than 25 years of automotive electronics design expertise and is a market leader in several automotive areas. We offer a broad scope of automotive-dedicated technologies such as BCDMOS, BCD-on-SOI and non-volatile CMOS, and automotive- qualified fabs (ISO 9001:2000, ISO TS 16949:2002, ISO 14001:2004).</p><ul><li><strong>Extensive Portfolio —</strong> The Atmel extensive portfolio ranges from car radio devices, ICs for standard body electronics, including high-temperature, LIN/CAN and motor driver ICs, solutions for car access systems, RKE/immobilizer and key fobs to our Atmel AVR® automotive microcontrollers, memories, capacitive touch and customer-specific ICs.</li><li><strong>Advanced Touch Solutions —</strong> From the proximity sensor for keyless door entry to the touchscreen of the GPS system, touch interfaces give the driver and passengers a sense of luxury, quality, and differentiation. With the industry's most comprehensive portfolio, Atmel provides a touch solution for virtually any automotive microcontroller.</li><li><strong>Development Support —</strong> Design-ins are supported by our application engineers, demonstration and evaluation kits, reference boards, software and detailed documentation.</li></ul>
