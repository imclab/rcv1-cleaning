<p><a href="http://www.autoguide.com/auto-news/wp-content/uploads//2013/01/VW-7-seater-crossover.jpg"><img class="aligncenter size-full wp-image-175888" title="VW-7-seater-crossover" src="http://www.autoguide.com/auto-news/wp-content/uploads//2013/01/VW-7-seater-crossover.jpg" alt="" width="646" height="396" /></a><strong>A lone sketch has leaked out ahead of the <a href="http://www.autoguide.com/auto-shows/2013-detroit-auto-show-coverage-2328.html">North American International Auto Show</a>, revealing the next step in <a href="http://www.autoguide.com/auto-news/category/manufacturers/volkswagen">Volkswagen&#8217;s</a> carefully planned growth strategy in America.</strong></p>
<p><span id="more-175887"></span></p>
<p>After expanding its market share considerably with new Passat and Jetta models designed specifically for the U.S., the German automaker is now planning a seven-seat crossover to rival the likes of the Ford Explorer, Nissan Pathfinder and Toyota Highlander. This new three-row machine hits a popular segment that Volkswagen is currently not competing in.</p>
<p style="text-align: center;"><strong>SEE MORE:</strong> <a href="http://www.autoguide.com/auto-shows/2013-detroit-auto-show-coverage-2328.html">2013 Detroit Auto Show Preview</a></p>
<p>While larger than the Touareg, the new 7-passeger crossover is expected to be priced well below that premium-level SUV, and be competitive with segment rivals.</p>
<p>Set to bow in Detroit on Monday, the new model will likely appear as a concept car rather than a production model, something the sketch hints at, especially with the crossover&#8217;s oversized wheels.</p>

    this.playerId = playerId;
    this.width = width;
    this.height = height;
    this.autoPlay = autoPlay;
    this.timer = 0;
    this.load = function() {
        if (!this.runNow()) {
            var self = this;
            this.timer = setInterval(function() { self.runNow(); }, 1000);
        }
    }
    this.runNow = function() {
        if (typeof xml_url != 'undefined') {
            if (this.timer)
                clearInterval(this.timer);
            callPlayer(xml_url, this.video, this.playerId, this.width, this.height, this.autoPlay);
            return true;
        }
        return false;
    }
}

            ajax_action: "getDivisions"
        }, function (data) {
            target.html('<option value="0">Select a Make</option>');
            jQuery.each(data, function (i, item) {
                target.append('<option value="' + item + '" >' + i + '</option>');
            });
            if (defaultDivision) {
                target.children('option[value="' + defaultDivision + '"]').attr('selected', 'selected');
            }
        });
    };
    this.ajaxGetModels = function(target, divisionID, modelYear, defaultModel) {
        jQuery.getJSON("http://www.autoguide.com/new-cars/newajax.php", {
            ajax_action: "getModels",
            divisionID: divisionID
            //modelYear:modelYear, 
        }, function (data) {
            target.html('<option value="0">Select a Model</option>');
            jQuery.each(data, function (i, item) {
                target.append('<option value="' + item.ModelID + '" >' + item.ModelYear + ' ' + item.ModelName + '</option>');
            });
            if (defaultModel) {
                target.children('option[value="' + defaultModel + '"]').attr('selected', 'selected');
            }
        });
    };
    this.ajaxGetGeoLocation = function(countryField, zipField, submitButton) {
        jQuery.getJSON("http://www.autoguide.com/new-cars/newajax.php?", {
            ajax_action: "getGeoLocation",
            rand: Math.random()
        }, function (data) {
            countryField.val(data.country_code);
            zipField.val(data.postal_code);
            
            if (data.country_code == 'CA') {
                submitButton.attr('src', "/images/icons/btn_price_quote_ca.gif");
            }
        });
    };
    this.ajaxGetUsedCars = function(level, makeField, modelField) {
        if (level == 0) {
            makeField.html("<option value=''>Loading...</option>");
        } else if (level == 1) {
            modelField.html("<option value=''>Loading...</option>");
        }
        jQuery.get("http://www.autoguide.com/used-cars/ajax.php", {
            ajax_action: "populate_lists",
            level: level,
            MakeVal: makeField.val()
        }, function (j) {
            var options = "";
            j = j.split(/,/);
            var level = j[0];
            var options = j[1];
            if (level == 0) {
                makeField.html(options);
                if (!jQuery.browser.msie) makeField.children("option:first").attr("selected", "selected");
                modelField.html("<option value=''>&nbsp;</option>");
            } else if (level == 1) {
                modelField.html(options);
                if (!jQuery.browser.msie) modelField.children("option:first").attr("selected", "selected");
            } else {
                return;
            }
        });
    }
}
